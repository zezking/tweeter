# Tweeter Project

Tweeter is a simple, single-page Twitter clone.

This is a completed project based on the given code by Lighthouse Labs

## Getting Started

Clone Repo and Run

```
npm install
npm run local
```

visit in Browser

```
localhost:8080/
```
